verify_address
Verify any US address and return coordinates, flood zone, fire/wind/earthquake risk, census tract, residential flag, and match confidence.
suggest_address
Auto-complete a partial address. Returns up to 10 candidates for the agent to present or disambiguate.
reverse_geocode
Look up the nearest verified US address for a lat/lon pair. Returns the matched address plus full enrichment.
get_coverage
Retrieve address counts and coverage tier (Full / Gap-fill / Partial) for all 50 states and territories.

Client Configuration

Add to ~/Library/Application Support/Claude/claude_desktop_config.json

{ "mcpServers": { "geoclear": { "type": "http", "url": "https://geoclear.io/mcp", "headers": { "X-Api-Key": "YOUR_API_KEY" } } } }

Add to ~/.cursor/mcp.json

{ "mcpServers": { "geoclear": { "type": "http", "url": "https://geoclear.io/mcp", "headers": { "X-Api-Key": "YOUR_API_KEY" } } } }

Any client that supports the MCP Streamable HTTP transport:

POST https://geoclear.io/mcp
# Required headers on every request: Content-Type: application/json X-Api-Key: YOUR_API_KEY Accept: application/json, text/event-stream # For subsequent requests in a session: Mcp-Session-Id: <id returned in first response>

Supports MCP protocol version 2025-03-26 (Streamable HTTP). Session lifecycle: POST to initialize, GET for SSE notifications, DELETE to terminate.

Walletless payment — x402

AI agents with a USDC wallet on Base can call the MCP endpoint without an API key. The server returns HTTP 402 with a payment requirement; the agent pays $0.004 USDC per session and proceeds. No signup, no rate limits beyond the per-call price.

# x402 flow — agent with USDC wallet on Base mainnet # 1. First request (no key, no payment) → server responds 402 POST https://geoclear.io/mcp Content-Type: application/json ← HTTP 402 PAYMENT-REQUIRED: <base64 JSON with USDC amount + recipient> # 2. Agent signs EIP-3009 transfer authorization and retries POST https://geoclear.io/mcp Content-Type: application/json payment-signature: <base64 JSON signed by wallet> ← HTTP 200 PAYMENT-RESPONSE: <settlement tx hash> Mcp-Session-Id: <session — no further payment needed>
Pricing logic: Free tier subscribers pay ~$0.0002/call effective. x402 callers pay $0.004/session — a 20× premium for zero-commitment, zero-signup access. Settlement via Coinbase's x402 facilitator (gasless EIP-3009 USDC transfer, Base mainnet).
