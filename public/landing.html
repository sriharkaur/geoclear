Capabilities

Everything a modern address API should do

From simple verification to full demographic enrichment — one API, one key, one bill.

✅
Address Verification
Confirm an address exists in the NAD master dataset. Returns 0–100 confidence score based on placement accuracy, field completeness, and anomaly flags.
🔍
Fuzzy Search
Typo-tolerant address matching via Levenshtein distance ranking. "Pensylvania Av" still finds "Pennsylvania Ave." Works with partial addresses.
⚡
Autocomplete
Sub-100ms typeahead suggestions as users type. Scoped by state or ZIP. Integrates into any address input form with 3 lines of JavaScript.
🏛
County FIPS Codes
Full 5-digit Federal county FIPS on every address. Built from Census Bureau's complete 3,234-county lookup table. No join, no extra cost.
📊
Census Tract
Census tract, block group, and GEOID via the Census Bureau's official geocoder. Required for HMDA compliance, CRA analysis, and demographic targeting.
🌊
FEMA Flood Zone
Real-time FEMA NFHL flood zone designation (AE, X, AH, VE…) and SFHA flag via FEMA's official map service. Required for mortgage underwriting.
📦
Bulk Verification
Verify up to 1,000 addresses in a single POST request. Returns full enrichment for each. Ideal for CRM cleanup, list hygiene, and batch import pipelines.
📍
Proximity Search
Find all addresses within N km of a lat/lon point. Bounding-box search with configurable radius. Returns full enrichment including FIPS and confidence.
🌐
Geographic Hierarchy
Full drill-down from country → state → county → city → ZIP → address. Every level returns address counts, FIPS, and metadata for spatial analysis.

const res = await fetch(
  'https://geoclear.io/api/address?street=Pennsylvania+Ave&city=Washington&state=DC&number=1600',

);
const { data } = await res.json();
// data[0] →
// { verified: true, confidence: 97, full_address: '1600 Pennsylvania Avenue NW',
//   city: 'Washington', state: 'DC', zip_code: '20500',
//   fips: '11001', timezone: 'America/New_York',

import requests

res = requests.get(
    'https://geoclear.io/api/address',
    params={'street': 'Pennsylvania Ave', 'state': 'DC', 'number': '1600'},
    headers={'X-Api-Key': 'nad_your_key_here'}
)
address = res.json()['data'][0]
print(address['fips'], address['confidence'])  # 11001  97
curl -H "X-Api-Key: nad_your_key_here" \
